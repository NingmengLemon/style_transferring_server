# 基于图像风格迁移的个性化数字艺术创作系统设计与实现

主要内容：用户上传自己的照片或绘画作品作为内容图像，选择梵高、毕加索等不同艺术家风格的图像作为风格图像；系统利用基于 VGG 网络的神经风格迁移算法，将风格图像的艺术风格迁移到内容图像上；提供参数调整功能，用户可自定义风格强度、细节保留程度等；生成的艺术作品支持在线预览、下载高清图片，还可分享至社交平台。
具体要求：使用 WikiArt 图像数据集（<https://www.wikiart.org/>）；基于 PyTorch 框架；生成图像的视觉效果符合用户选择风格的比例不低于 80%，单张图像风格迁移时间小于 2.5 秒。暴露出HTTP端口供外部应用使用.
